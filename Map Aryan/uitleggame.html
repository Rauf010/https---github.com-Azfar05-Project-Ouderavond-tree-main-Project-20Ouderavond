Uitleg Game Aryan


hoe werkt de game

kort samengevat bij deze game moet je proberen zo vaak mogelijk over blokjes te springen.
deze game is gebaseerd op de offline google game die t-rex mini game heet.
ik besloot ook om deze game te maken omdat het heel simpel is en omdat ik nog niet zo goed ben in games programmeren

Game Handleiding

  1. gebruik de space bar knop om te springen
  2. spring over de blauwe blokjes
  3. als je het blokje raakt dan ben je af
  4. als je af bent kun je je score zien
  5. probeer je highscore te verbeteren en veel plezier met de game
