Reflectie op mijn competenties

Mijn naam is Rauf Bhikhie, ik ben een software developer in opleiding. Ik studeer dit vak op het Techniek College Rotterdam.
Hier heb ik veel geleerd over programmeren met verschillende codeertalen en hoe deze toegepast kunnen worden.
Tot nu toe heb ik de basis van HTML, CSS, JavaScript en PHP geleerd. Ook heb ik extra vaardigheden geleerd die goed van pas kunnen komen.
Ik heb geleerd designs (wireframes) te maken van elk project waar ik aan begin, een database te sorteren en een schets maken van hoe de webpagina's gelinkt worden (sitemap).
Ondanks het feit dat ik dit geleerd heb, ben ik weer een beetje roestig wil ik mijn vaardigheden weer toepassen, dit komt door te weinig herhalen.
Dit is dan ook een verbeterpunt voor mij. Als ik blijf herhalen heb ik de potentie om ooit een professional te worden als software developer.
